# Baseball Domain Knowledge

> 야구 도메인 로직 검증을 위한 규칙 및 체크리스트

## 도메인 지식 범위

### 1. 경기 규칙
- 타자/주자/수비 규칙
- DH(지명타자) 규칙 및 해제 조건
- 대타/대주자 규칙
- 이닝/경기 종료 조건

### 2. 기록 규칙
- 타격 기록 (안타, 타점, 득점 등)
- 투구 기록 (자책점, 승패, 세이브 등)
- 수비 기록 (자살, 실책, 병살 등)

### 3. 특수 상황
- 인필드 플라이
- 보크
- 주루 방해/수비 방해
- 콜드게임/서스펜디드 게임

## DH (Designated Hitter) 규칙

### DH 지정 조건
- 선발 투수 등판 전에 DH 지정 필수
- 타순표 제출 시 명시되어야 함
- DH는 투수를 위해 타격만 수행

### DH 해제 조건
```
DH 해제 트리거:
1. 투수가 다른 포지션(야수)으로 이동
2. 투수가 DH 타순에 들어가 타격
3. DH 선수가 수비 위치에 투입
```

### DH 해제 후 처리
- DH가 해제되면 투수가 타순에 포함
- 재지정 불가 (경기 종료까지 유지)
- 타순 재배열 불가

## 타격 기록 규칙

### 안타 판정
- 타구가 야수의 정규 수비 범위를 벗어나 안전하게 출루
- 야수의 실책이 없어야 안타로 인정
- 내야안타: 수비수의 송구 전 1루 도달

### 타점 규칙
```
타점 부여 조건:
1. 안타로 주자 득점
2. 희생플라이로 주자 득점
3. 내야 땅볼로 주자 득점 (실책 없음)
4. 볼넷/사구로 밀려나가 득점

타점 불인정:
- 병살타로 득점
- 야수 실책으로 득점
```

### 희생타/희생플라이
- 희생번트: 주자 진루 목적의 번트 (타수 제외)
- 희생플라이: 외야 플라이로 주자 득점 (타수 제외, 타점 인정)
- 조건: 2아웃 미만, 주자가 실제로 진루/득점

## 투수 기록 규칙

### 승리투수 자격
```
선발 투수:
- 5이닝 이상 투구
- 교체 시점에 팀이 리드
- 경기 종료까지 리드 유지

구원 투수:
- 1아웃 이상 기록
- 교체 시점에 팀이 리드하거나 동점
- 경기 종료까지 리드 유지
```

### 자책점 계산
- 실책/방해/패스트볼이 없었다면 방지할 수 있던 실점만 자책점
- 실책으로 연장된 이닝의 실점은 비자책
- 투수 자신의 실책은 자책점으로 계산

### 세이브 조건
```
세이브 부여 조건 (모두 충족):
1. 구원 등판 (선발 아님)
2. 승리 투수 아님
3. 3점 이하 리드 상황에서 1이닝 이상 투구
   OR 등판 시 동점 주자가 베이스에 있음
   OR 3이닝 이상 무실점 투구
4. 경기 종료까지 리드 유지
```

## 특수 상황 규칙

### 인필드 플라이
- 조건: 주자 1,2루 또는 만루 + 2아웃 미만
- 내야 뜬공을 심판이 "인필드 플라이" 선언
- 타자 즉시 아웃, 주자는 진루 의무 없음

### 보크
```
보크 상황:
- 투구 동작 중 중단
- 견제구를 던지지 않고 모션만
- 투구판 접촉 없이 투구
- 주자 없는 상황에서는 보크 없음 (볼 선언만)

보크 판정 시:
- 모든 주자 1개 베이스 진루
- 타자는 타석 유지
```

## 프로젝트 Entity 매핑

검증 시 아래 Entity 메서드가 이 문서의 규칙과 일치하는지 확인:

### Game Entity (경기 진행)
- `Game.start()`, `Game.finish()` — 경기 시작/종료
- `Game.cancel(reason)`, `Game.callGame(reason)` — 취소/콜드게임
- `Game.forfeit(winnerTeamId, reason, teams)` — 몰수패
- `Game.postpone(newScheduledAt, reason)`, `Game.reschedule(newScheduledAt)` — 연기/일정변경
- `Game.recordOut(): Boolean` — 아웃 기록, `Game.advanceBatter()` — 타순 진행
- `Game.setRunner(base, playerId)` — 주자 설정
- `Game.addBall(): Boolean`, `Game.addStrike(): Boolean` — 볼/스트라이크 카운트
- Computed: `currentInningDisplay`, `isExtraInning`

### BattingRecord / PitchingRecord (기록)
- `PlateAppearanceResult` enum — 타석 결과 (안타, 삼진, 볼넷 등)
- `PitchResult` enum — 투구 결과
- `PitchingDecision` enum — 승/패/세이브/홀드

### LineupEntry / LineupSubmission (라인업)
- `LineupValidator` — DH 규칙 포함 라인업 검증
