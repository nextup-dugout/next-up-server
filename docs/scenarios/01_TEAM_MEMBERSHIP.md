# 🧢 도메인 시나리오 01: 팀 & 멤버십 (Team & Membership)

> **"술자리 투표로 뽑던 감독은 이제 그만. 데이터와 시스템으로 운영되는 진짜 야구팀."**

---

### 💥 결정적 차이 (The Game Changer)

| 상황 | 기존 방식 (AS-IS) 😫 | NEXT-UP 방식 (TO-BE) ⚡ |
| :--- | :--- | :--- |
| **팀 가입** | 카페 등업 신청하고 **3일간** 운영진 승인 대기. (읽씹의 고통) | 조건 맞으면 **즉시 신청**, 앱 푸시 알림으로 **실시간 승인**. (3분 컷) |
| **기록 증명** | "저 고등학교 때 좀 했습니다" (말로만 주장 ➔ **검증 불가**) | **"통산 타율 0.412"** 데이터 카드 자동 첨부. (신뢰도 100%) |
| **감독 선출** | 총무가 탈퇴한다고 협박해야 **술자리에서 억지로** 누군가 떠맡음 | 기간 정해놓고 앱에서 **익명 비밀 투표**. 1위에게 **시스템 권한 자동 이양**. |
| **팀 운영** | 감독 혼자 회비 관리, 라인업 독재 ➔ **불만 폭발 및 팀 해체** | 투명한 회계/출석/성적 데이터 공개 ➔ **시스템에 의한 민주적 운영**. |
| **진상 처리** | 회비 안 내고 버티는 회원... **말하기 껄끄러워서 방치** | **[미납자 활동 제한]** 기능으로 라인업 등록 자동 차단. (금융치료) |

## 👤 User Scenario: 민주적인 팀 리빌딩

### Scene 1. 감독이 도망갔다 (비상 상황)
해머스 팀 감독님이 개인 사정으로 갑자기 잠수를 탔습니다. 팀 단톡방은 혼란에 빠집니다.
*   "이번 주 오더 누가 내요?"
*   "회비 통장 비번 아시는 분?"

### Scene 2. 시스템이 주도하는 선거 (Rebuilding)
남은 운영진이 앱에서 **[비상대책위원회 모드]**를 켭니다.
1.  **선거 개설**: 시스템이 자동으로 **[차기 감독 선출 투표]**를 생성하여 전원에게 알림을 보냅니다.
    *   알림: 🚨 *감독 부재로 비상 선거가 시작되었습니다. (48시간 내 투표 필수)*
2.  **공정한 입후보**: 평소 팀 살림을 챙기던 '박총무' 님이 입후보 버튼을 누릅니다.
    *   공약 등록: "투명한 회비 공개! 전원 2이닝 이상 출전 보장!" 트렌디한 공약 카드가 피드에 뜹니다.
3.  **익명 투표**: 눈치 볼 필요 없이 앱에서 '박총무'를 찍습니다. 누가 찍었는지는 절대 모릅니다.

### Scene 3. 깔끔한 권한 이양 (Transfer)
투표 마감 1초 후.
1.  **결과 발표**: 📱 *[선거 결과] 박총무 님이 85% 득표율로 제 3대 감독에 당선되었습니다!*
2.  **자동 승계**:
    *   박총무 앱 화면: 일반 회원 UI ➔ **[감독 전용 대시보드]**로 즉시 변경되어 '강퇴', '공지 작성' 버튼이 생깁니다.
    *   기존 도망간 감독: **[관리자]** 권한 자동 박탈 및 일반 회원으로 강등되어 더 이상 팀을 건드릴 수 없습니다.

## 👤 User Scenario: 새로운 선수 영입 (Recruiting)

### Scene 1. 데이터 기반 스카우팅
팀 성적이 바닥을 기고 있습니다. 유격수가 구멍입니다.
1.  **모집 공고**: 감독이 [팀원 모집] 설정을 켭니다.
    *   조건: "서울 거주", "유격수 가능", "3부 리그 경험자"
2.  **지원자 도착**: 김신입 선수가 지원했습니다.
    *   **프로필 카드**: 나이, 사는 곳은 기본. **[최근 3년 통산 타율 그래프]**와 **[주 포지션 수비율]**이 뜹니다.
    *   "오, 3년 전에는 잘했는데 작년에 좀 쉬었네? 그래도 기본기는 있겠다."

### Scene 2. 가입 승인과 온보딩
1.  **등번호 전쟁**: 김신입 선수가 '7번'을 달라고 했지만 이미 주장님이 쓰고 있습니다.
    *   승인 시 팝업: *"7번은 사용 중입니다. 다른 번호를 부여할까요?"*
    *   감독이 '77번'으로 수정해서 승인합니다.
2.  **웰컴 패키지**: 김신입 선수 폰에 🎊 *[해머스 입단을 환영합니다]* 알림이 뜨고, 팀 전용 '일정 캘린더'와 '게시판' 접근 권한이 **자동으로 풀립니다**. (따로 초대할 필요 없음)

### Scene 3. 아름다운 이별 (Ban)
회비를 6개월째 안 내고 연락 두절인 유령 회원.
1.  **원클릭 정리**: 감독이 멤버 관리 목록에서 [내보내기]를 누릅니다.
    *   사유 선택: "장기 미활동 및 회비 미납"
2.  **재가입 방지**: *"이 사용자의 재가입을 영구적으로 차단하시겠습니까?"* 체크박스에 체크합니다.
    *   이제 이 회원은 팀을 검색해도 [가입 신청] 버튼이 비활성화됩니다. 깔끔합니다.

## 💡 핵심 기능 요약

### 1. 역할 기반 접근 제어 (RBAC)
*   **감독(Owner)**: 팀 삭제, 정보 수정, 관리자 임명권 (절대 권력).
*   **운영진(Manager)**: 가입 승인, 라인업 제출, 회계 관리.
*   **일반(Member)**: 투표 참여, 게시글 작성.
*   **게스트(Guest)**: 투표 불가능, 경기 기록만 쌓임.

### 2. 팀 브랜딩 커스텀
*   **엠블럼 제작소**: 디자이너가 없어도 앱 내에서 [방패 모양] + [배트 아이콘] + [빨간색] 조합으로 **10초 만에 그럴싸한 팀 로고**를 만들 수 있습니다.

### 3. 상태 관리 자동화
*   팀원이 탈퇴 버튼을 누르면 즉시 **'탈퇴 회원'**으로 분류되고, 과거 기록은 **'은퇴 선수'** 기록으로 보존되어 팀 역사(History) 페이지에 남습니다. (기록은 사라지지 않음)
