# ⚾️ 도메인 시나리오 02: 경기 준비 & 라인업 (Match Prep & Lineup)

> **"매주 수요일의 '복붙 투표' 지옥, 이제 안녕."**

---

### 💥 결정적 차이 (The Game Changer)

| 상황 | 기존 방식 (AS-IS) 😫 | NEXT-UP 방식 (TO-BE) ⚡ |
| :--- | :--- | :--- |
| **참석 투표** | 단톡방 공지에 **"1.홍길동 2.김철수..."** 댓글 복사 붙여넣기 노가다 | 푸시 알림 와서 **[참석] 버튼 한 번 누르면 끝**. (복붙 금지) |
| **인원 파악** | 총무가 댓글 세어보고 엑셀에 따로 정리해야 함 | **"참석 9, 불참 2, 미정 3"** 실시간 그래프가 바로 뜸 |
| **독려(Nudge)** | 안 뽑은 사람 찾아서 "투표 좀 해주세요" **일일이 멘션** | **[미투표자 재촉하기]** 버튼 하나면, 안 한 사람 폰만 징징 울림 |
| **라인업 작성** | 노트 찢어서 볼펜으로 적고(악필 주의), **폰으로 사진 찍어** 단톡방 올림 | 불참자 자동 필터링 된 화면에서 **드래그 앤 드롭**. |
| **오더지 제출** | 경기장 가서 기록원에게 종이 오더지 내고, **이름 틀려서 또 수정**하고... | 감독이 집에서 **[제출]** 누르면 **기록실 태블릿에 자동 세팅**. (종이 필요 X) |

## 👤 User Scenario: 카톡 지옥 탈출기

### Scene 1. 투표 알림이 왔다 (User Experience)
수요일 점심시간, **카톡이 아닙니다.** NEXT-UP 앱 푸시가 옵니다.
*   🔔 *[이번 주말 경기 참석 가능하신가요?]*
*   잠금 화면을 꾹 눌러(Long Press) **[🙆🏻‍♂️ 참석]** 버튼을 누릅니다. **(소요 시간: 1초)**
*   단톡방 알림 꺼놔서 공지 못 봤다는 핑계가 이제 안 먹힙니다.

### Scene 2. 감독의 쾌적한 금요일 밤 (Admin Experience)
금요일 밤, 술 한잔 하다가 감독님이 폰을 켭니다.
1.  **Nudge(쿡 찌르기)**: 아직 '미정'인 게으른 팀원 3명이 보입니다. [재촉하기] 버튼을 누릅니다.
    *   해당 3명 폰에만 🚨 *감독님이 애타게 기다립니다!* 알림이 가서 즉시 응답을 받아냅니다.
2.  **스마트 라인업**:
    *   화면에 **참석자 10명 카드만 활성화** 되어 있습니다. (불참자 실수로 넣을 일 0%)
    *   4번 타자 자리에 '이거포'를 쓱(Drag) 끌어다 놓습니다.
    *   유격수에 '김신입'을 놓으려는데 앱이 말해줍니다. 💡 *이 선수는 최근 2루수로 더 많이 뛰었습니다.*
    *   "아 맞다." 하고 2루수로 옮깁니다(Drop).

### Scene 3. 경기 직전의 전력분석 (Strategy)
일요일 아침, 상대팀 라인업이 1시간 전에 공개되었습니다.
1.  **적을 알라**: 감독이 상대팀 오더를 앱으로 봅니다.
    *   "어? 쟤네 에이스 투수가 오늘 4번 타자로도 나오네?"
    *   상대 1번 타자의 **[핫존(Hot Zone)]** 데이터를 봅니다. 바깥쪽 공에 약점이 있습니다.
2.  **전략 수정**: 우리 선발 투수에게 카톡을 보냅니다. "오늘 1번 타자한테는 바깥쪽 승부만 해라."
    *   데이터가 있으니 막연한 똥촉으로 승부하지 않습니다.

## 💡 핵심 기능 요약

### 1. 투표 상태 대시보드
*   **참석/불참/미정**: 3가지 상태를 도넛 차트로 보여줍니다.
*   **불참 사유 수집**: "가족모임", "부상", "출근" 등 사유를 선택하게 하여, 다음 경기 참석 가능성을 예측합니다.

### 2. 포지션 밸런스 체크 (Validation)
*   라인업을 짰는데 **"포수"** 포지션이 비어있으면 [제출] 버튼이 안 눌립니다.
*   한 선수를 1루수와 지명타자(DH) 두 곳에 동시에 넣으면 🔴 **[중복 배치]** 경고가 뜹니다. 실수할 수가 없습니다.

### 3. 오더지 공유 시스템
*   감독이 제출하는 순간, 아래 3곳으로 동시 전송됩니다.
    1.  **팀원들**: "나 오늘 몇 번 타자지?" 확인.
    2.  **상대팀**: (경기 1시간 전) 상호 라인업 교환 룰 적용.
    3.  **기록실**: 기록원 태블릿에 데이터 로딩. (기록원이 수기로 타이핑 안 해도 됨)
