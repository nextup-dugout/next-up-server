### 💥 결정적 차이 (The Game Changer)

| 상황 | 기존 방식 (AS-IS) 😫 | NEXT-UP 방식 (TO-BE) ⚡ |
| :--- | :--- | :--- |
| **경기 찾기** | 커뮤니티 게시판 **F5(새로고침) 무한 반복**. "이미 구했나요?" 문자 돌리기 | 내가 원하는 조건(포수, 무료) 등록해두면 **매칭 시 푸시 알림 옴** |
| **참여 신청** | "저 3부 경험 있고요..." 구구절절 문자로 어필하다가 읽씹 당함 | 내 **[용병 프로필 카드]**(평점, 기록 포함) **원클릭 전송**. (신뢰도 ↑) |
| **기록 관리** | 남의 팀 기록지에 이름 대충 적고 끝. **잘 쳐봤자 기록 증발(삭제)** | 내 계정으로 로그인해서 뛰기 때문에 **타율/수비 기록이 내꺼로 영구 저장** |
| **먹튀/노쇼** | "아침에 못 간다고 문자 띡 옴" ➔ 차단 말고는 방법이 없음 | **노쇼 3회 시 계정 영구 정지**. 확실한 **블랙리스트/평점 시스템** 가동 |

## 👤 User Scenario: 신뢰 기반 매칭 혁명

### Scene 1. 5분 대기조 탈출 (Guest Experience)
주말 아침, 갑자기 야구가 하고 싶은 나용병 님.
1.  **스마트 알림**: 게시판을 뒤지는 대신 아침밥을 먹습니다. 📱 띠링!
    *   *[추천] 집 근처(3km) 해머스 팀에서 '내야수'를 급구합니다! (참가비 무료)*
2.  **원클릭 지원**: [신청하기] 버튼을 누릅니다.
    *   내 프로필(용병 평점 4.9점)이 감독에게 전송됩니다. 구차하게 "저 펑크 안 냅니다" 말할 필요가 없습니다. 제 평점이 증명하니까요.

### Scene 2. 감독의 안심 채용 (Host Experience)
발등에 불 떨어진 해머스 감독.
1.  **인재 검증**: 신청자가 3명이나 왔습니다. 프로필을 비교합니다.
    *   후보 A: 평점 2.5점 (노쇼 이력 1회) ➔ **[거절]**
    *   후보 B: 평점 5.0점 (최근 5경기 타율 0.450) ➔ **[수락]**
2.  **자동 마감**: 수락하자마자 공고는 '마감' 처리되고, B선수에게 오시는 길 안내가 자동 발송됩니다.

### Scene 3. 기록의 자산화 (Asset Experience)
경기가 끝났습니다. 3타수 2안타를 쳤네요.
1.  **커리어 누적**: 집에 와서 [내 기록실] > [용병 탭]을 봅니다.
    *   오늘 기록이 합산되어 **'용병 통산 타율'**이 올랐습니다.
2.  **스카우트 제의**: 내 기록을 본 다른 팀 감독이 [입단 제의] 메시지를 보냅니다.
    *   "저희 팀 주전 유격수로 모시고 싶습니다."
    *   일회성 용병 활동이 **스카우트의 기회**로 연결됩니다.

## 💡 핵심 기능 요약

### 1. 매너 온도 (Trust Score)
*   **당근마켓 스타일**: 경기가 끝나면 서로 평가를 합니다.
    *   용병 ➔ 팀: "텃세 없고 친절해요."
    *   팀 ➔ 용병: "시간 약속 잘 지키고, 장비도 잘 챙겨오셨어요."
*   이 평가는 다른 팀이 나를 뽑을 때 1순위 판단 기준이 됩니다.

### 2. 에스크로 결제 (Safe Payment)
*   **참가비 먹튀 방지**: 유료 용병 게임(2만원)인 경우, 돈을 감독 계좌로 바로 쏘지 않습니다.
*   **플랫폼 보관**: 결제한 돈은 NEXT-UP이 보관하다가, **경기 종료 확인 후** 감독에게 입금됩니다.
*   **100% 환불**: 우천 취소되거나 감독이 일방적으로 취소하면 **자동으로 환불**됩니다. (전화해서 돈 돌려달라고 싸울 필요 없음)

### 3. 용병 전용 기록실
*   내 소속팀 기록과 용병 기록을 **분리해서도, 합쳐서도** 볼 수 있습니다.
*   "나는 용병으로 뛸 때 더 잘한다"는 지표를 볼 수 있는 재미있는 통계를 제공합니다.
