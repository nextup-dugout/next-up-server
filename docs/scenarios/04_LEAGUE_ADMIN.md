> **"대진표 짜느라 3일 밤새던 사무국장님, 이제 클릭 몇 번으로 끝내세요."**

---

### 💥 결정적 차이 (The Game Changer)

| 상황 | 기존 방식 (AS-IS) 😫 | NEXT-UP 방식 (TO-BE) ⚡ |
| :--- | :--- | :--- |
| **일정 편성** | 엑셀 켜놓고 달력 보며 **수작업 매칭**. 충돌 나면 처음부터 다시 짬. | 구장/팀 조건 입력하면 **자동 배정 알고리즘**이 초안을 만들어줌. |
| **순위 확인** | 홈페이지 접속 ➔ 게시판 클릭 ➔ **hwp/xls 파일 다운로드** ➔ 폰에서 깨짐 | 앱 켜자마자 **실시간 랭킹 확인**. (승차, 승률 포함) |
| **우천 취소** | 비 오면 전화 돌리고, **밀린 일정 엑셀에서 다시 끼워넣기** (멘붕) | **[우천 순연]** 버튼 클릭 시, 남은 빈 시간대로 **자동 이동** |
| **개인 타이틀** | "내가 타율 1위 맞아?" 확인하려면 **전체 기록 뒤져봐야 함** | 내 프로필에 **[🥇 리그 타격 1위]** 뱃지가 자동으로 붙음 (간지 폭발) |
| **이의 제기** | 협회 게시판에 글 쓰고 **비밀번호 까먹어서 수정 못함** | 해당 타석 기록 **버튼 하나로 [정정 신청]**. 영상 첨부 가능. |
| **운영 업무** | 사무국장이 **주말 밤 새서** 수기 입력 및 계산 | **'경기 종료' 버튼 누르면 0.1초 만에** 리그 홈페이지 전체 자동 갱신 |

## 👤 User Scenario: 효율적인 운영 시스템

### Scene 1. 대진표 짜기 탈출 (Scheduling Experience)
매년 2월, 시즌 개막을 앞둔 사무국장님의 책상은 달력과 메모지로 엉망입니다.
*   이 팀 저 팀 시간 맞춰주다가 머리가 터집니다.

**하지만 NEXT-UP에서는?**
1.  **기본틀 만들기**:
    *   **참가팀**: 10팀 (풀리그 = 팀당 9경기, 총 45경기)
    *   **구장 시간표**: "매주 일요일 08:00, 10:00, 12:00, 14:00, 16:00 (5타임)" 템플릿 등록.
2.  **자동 배정 실행**: **[일정 자동 채우기]** 버튼을 누릅니다.
    *   시스템이 **라운드 로빈(Round Robin) 규칙**에 따라 1주차부터 9주차까지 대진표를 빈 시간대에 순서대로 채워 넣습니다. (단순 로직 이용)
3.  **마우스로 쓱쓱 조정**:
    *   "아, 해머스 팀은 첫 경기(08시) 좀 피해달랬지."
    *   화면에서 해머스 경기 블록을 마우스로 잡고(Drag), 오후 2시 빈 슬롯으로 옮깁니다(Drop).
    *   중복 체크: 이미 그 시간에 다른 경기가 있으면 빨간색으로 **[시간 겹침]** 경고가 떠서 실수를 막아줍니다.

### Scene 2. 비 오는 날의 평화 (Rain Cancle)
장마철 일요일 아침 6시, 비가 쏟아집니다.
1.  **일괄 취소**: 관리자가 앱에서 **[금일 경기 전체 취소]** 버튼을 누릅니다.
    *   **공지 자동 발송**: 오늘 예정된 선수들에게 동시에 🌧️ *[우천 취소 알림]*이 갑니다.
2.  **일정 미루기**: 취소된 경기들을 선택해서 **[잔여 일정으로 이동]**을 누릅니다.
    *   시즌 맨 뒤의 **'예비일'**혹은 중간에 비어있는 타임으로 일정들이 그대로 복사되어 이동합니다.

## 👤 User Scenario: 고품격 리그 정보 서비스

### Scene 3. 숨 막히는 순위 싸움 (Team Experience)
리그 마지막 경기를 앞둔 해머스 팀.
1.  **경우의 수 계산**: 리그 페이지에 승점표와 남은 경기 수가 한눈에 보입니다.
2.  **실시간 변동**: 라이벌 팀 경기가 먼저 끝났습니다.
    *   라이벌 팀 패배 ➔ 우리 팀 순위가 3위에서 **2위(플레이오프 직행)**로 실시간 점프합니다.
    *   단톡방이 난리가 납니다. "야! 쟤네 졌다! 우리 이기면 우승이다!"

### Scene 4. 타이틀 홀더의 자부심 (Player Experience)
타격왕 경쟁 중인 김신입.
1.  **리더보드**: [개인 순위] 탭에서 **타율 부문**을 누릅니다.
    *   1위: 김신입 (0.512)
    *   2위: 박라이벌 (0.508)
2.  **동기 부여**: "오늘 안타 하나만 더 치면 1위 굳히기 가능" 메시지가 뜹니다.

### Scene 5. 이의 제기 처리는 쿨하게 (Correction)
1.  **민원 발생**: 선수가 앱에서 "3회초 안타가 에러로 기록됐어요"라며 **[정정 신청]**을 보냅니다.
2.  **원터치 처리**: 기록위원장 폰에 알림이 뜹니다. 영상을 보니 안타가 맞습니다.
    *   **[승인]** 버튼 클릭.
    *   해당 기록이 수정되고, 리그 타율 순위표까지 연쇄적으로 자동 수정됩니다. (게시판 댓글로 싸울 필요 없음)

## 💡 핵심 기능 요약 (Admin Features)

### 1. 자동 배정 도구 (Simple Auto-Assign)
*   **라운드 로빈 생성기**: 팀 수만 입력하면 "누가 누구랑 붙어야 하는지" 전체 대진 조합(Match-up)을 수학적으로 계산해줍니다.
*   **슬롯 채우기**: 계산된 매치업을 구장 시간표 빈칸에 순서대로 꽂아줍니다.
*   **충돌 감지**: 수동으로 옮길 때 "같은 팀이 같은 시간에 두 번 뛸 수 없음" 같은 기본 물리적 충돌만 막아줍니다.

### 2. 몰수패 및 징계 관리
*   **버튼 하나로 몰수패**: 점수(7:0) 자동 입력 및 승패 처리.
*   **부정선수 체크**: 라인업에 등록되지 않은 선수는 입력 자체가 안 되게 막아서 원천 봉쇄.

### 3. 미납 관리 (Financial)
*   **리그비 체크**: 대시보드에서 각 팀 이름 옆에 [납부완료] / [미납] 태그 관리.
